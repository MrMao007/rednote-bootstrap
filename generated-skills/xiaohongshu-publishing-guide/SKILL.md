### 一、发布前准备

**置信度：** 高（被 5 篇内容交叉验证）

**1. 账号定位与基础设置**

在发布第一篇笔记之前，需要完成以下准备：

- **明确定位**：问自己三个问题——擅长什么？想通过小红书做什么？目标用户是谁？定位不要太细，要让用户一眼看出关注你能获得什么价值。
- **完善个人资料**：头像、昵称、简介要统一专业，简介中清晰说明"你是谁、做什么、能提供什么"。
- **找对标账号**：找10个以上同赛道低粉爆文博主，拆解他们的封面布局、标题写法、文章结构。越多越好。
- **养号**：新号注册后，先用3-7天正常浏览、点赞、评论、收藏，让平台识别你的兴趣标签。不要注册后立刻发布内容。

**2. 选题与内容准备**

- **利用笔记灵感**：在APP首页点击左上角三条杠 → 创作者中心 → 全部服务 → 笔记灵感，可搜索当前热门话题。
- **选题方法**：热点蹭题（结合时事热点）、对标拆解（模仿爆款笔记结构）、用户痛点（从评论区挖掘需求）。
- **内容准备**：笔记要有深度、有知识、能引发共鸣。核心要素包括——人群定位、场景描述、痛点挖掘、解决方案、情感打动、行动步骤。

### 二、笔记发布完整流程（七步法）

**置信度：** 高（被 4 篇内容交叉验证）

**第一步：进入创作入口**

⚠️ **重要提示**：不要直接点底部的"+"号发布，这样发布的笔记流量会少很多。正确做法是：
1. 点击APP首页左上角三条杠（≡）
2. 选择「创作者中心」
3. 点击「全部服务」
4. 点击「笔记灵感」
5. 搜索或选择热门话题
6. 选好话题后点击「去发布」

**第二步：选择笔记类型与上传素材**

- **图文笔记**：最多上传 **18张图片**，建议精选6-9张
- **视频笔记**：选择 **1个视频**
- 素材要清晰、有质感，避免模糊或杂乱

**第三步：制作封面**

- 提前用修图软件（如醒图、Canva、黄油相机）制作一张有吸引力的封面
- 封面要有代表性，能引发用户点击欲望
- 保持封面风格统一，形成个人辨识度
- 目标受众的痛点或需求要在封面中直观体现

**第四步：编辑内容**

- 导入封面图后，编辑正文内容
- 合理安排图文布局，使内容更具吸引力
- 图文笔记重点看「滑动超3屏率」和完读率，建议内容有一定深度
- 视频笔记重点看「前3秒留存」，开头要抓人

**第五步：撰写标题**

- 标题要简洁明了，让用户一眼了解笔记内容
- 标题公式推荐：痛点 → 分析 → 效果（建议这个结构）
- 可自行添加文案分段 + 表情符号增加可读性
- 标题中融入关键词，提高搜索曝光

**第六步：添加标签与话题**

- 点击「添加标签」按钮，选择与内容相关的标签
- **最多可添加10个标签**，建议全部用满
- 添加热门关键词标签，提高笔记曝光度
- 探店类目记得添加地址/定位
- 选择相关话题参与，增加流量入口

**第七步：发布笔记**

- 检查无误后，点击「发布」按钮
- 发布后**不要立即退出**，停留在笔记页面观察
- 发布后2小时内是冷启动关键期，互动质量占流量分配权重60%

### 三、内容审核机制与平台规则（2026年最新）

**置信度：** 高（被 3 篇内容交叉验证）

**1. AI内容标注强制规则**

- 所有使用AI生成的内容（文案、图片、视频、配音）**必须标注**
- 标注路径：发布时在「高级选项 → 内容类型声明 → 内容来源说明」中勾选「AI辅助创作」
- **未标注后果**：首次违规限流7天，二次违规笔记下架，三次违规账号降权1个月
- 纯原创内容无需标注

**2. 原创与同质化内容规则**

- 非原创内容必须明确标注来源
- 洗稿、搬运、模板化批量生产内容流量减少90%+，严重者直接下架
- 同质化判定：同一账号发布3篇以上相似内容（换图不换文、换文不换图），或矩阵号内容高度雷同
- 平台鼓励真实体验 + 专业价值 + 情感共鸣的原创内容，深度内容（避坑指南、流程解析）获得流量倾斜

**3. 绝对化用语与虚假宣传禁令**

- 严禁使用绝对化词汇："最"、"第一"、"绝对"、"100%见效"、"根治"、"永不反弹"等
- 医美内容需双证备案（医师资格证+医疗机构执业许可证）
- 养生/健康类禁止承诺治疗效果（如"7天瘦10斤"、"祛斑"、"治疗痘痘"）
- 母婴类禁止推荐无资质产品，禁止使用"宝宝必买"、"智商税"等极端表述
- 虚假对比与测评，流量严格限制

**4. 导流与站外引流红线（2026年最严整治）**

- **绝对禁止**：评论区/私信区出现"私我"、"看主页"、"V我"、"加群"等，图片藏二维码、语音提微信，OCR直接识别封禁
- **合规引流方式**：仅通过官方私信获客工具、服务留资卡、企业名片等指定路径
- **违规后果**：首次导流笔记扣费+扣费，二次导流账号限流15天，三次导流永久封号

**5. CES评分系统（2026年权重更新）**

- 官方公式：CES = 点赞×1分 + 收藏×1分 + 评论×4分 + 转发×4分 + 关注×8分
- 核心变化：关注权重提升至点赞的8倍，评论/转发权重为点赞4倍
- 平台精准识别"有效互动"，机械求赞、刷评论、秒赞秒藏等异常互动会被扣分
- 深度互动直接决定流量层级

**6. 阶梯式处罚标准**

| 违规程度 | 处罚措施 | 典型行为 |
|---------|---------|---------|
| 轻微违规 | 限流7天+笔记限教数 | AI未标注、轻微导流暗示 |
| 一般违规 | 笔记下架+账号扣分 | 刷对比/虚假宣传、未报备合作 |
| 严重违规 | 账号降权1个月+限制功能 | 多次导流、批量发布低质内容 |
| 特别严重 | 永久封号+冻结账号 | 黑灰产、严重造假、多次严重违规 |

### 五、发布后运营要点

**置信度：** 高（被 4 篇内容交叉验证）

**1. 冷启动关键期（发布后2小时）**

- 发布后2小时内的互动质量占流量分配权重60%
- 让朋友/粉丝在评论区预览内容10-30秒后再互动
- 点赞、收藏、评论都有帮助，通过找到千人行列
- 前3秒留存是视频笔记冷启动核心

**2. 发布后自检清单**

- 检查笔记是否被正常收录（搜索标题关键词能否找到）
- 观察初始曝光数据（新号200-500曝光为正常基础）
- 如果小眼睛（浏览量）<100且无违规提示，进行自查

**3. 互动维护**

- 积极回复评论，提高互动率
- 评论区互动能有效提升CES评分（评论权重是点赞的4倍）
- 鼓励8字以上的长评论（平台识别为高质量互动）

**4. 数据复盘**

- 一周后分析：留存率、互粉率
- 关注维度：阅读量、互动量、转化数据
- 根据数据调整选题策略、封面设计、发布时间

### 六、导致限流的20个危险行为

**置信度：** 高（被 3 篇内容交叉验证）

**限流的两种类型：**
- **明限流**：收到平台违规通知，立即停止违规行为，发布优质笔记，通常能较快恢复
- **暗限流**：流量突然下降但无官方警告，需要逐项自查

**20个危险行为清单：**

1. **笔记含引流内容**：含二维码、微信号或其他平台引流信息
2. **使用违禁词语**：笔记中出现平台禁止的词汇
3. **发布非原创内容**：搬运、抄袭他人内容
4. **频繁删除或隐藏笔记**：系统判定账号内容质量不稳定
5. **账号质量低**：发布内容含纯无意义内容，质量明显下滑
6. **长期不更新**：一周不活跃即可能被判定为不活跃账号
7. **多次违规记录**：反复违反社区规范，账号信誉度受损
8. **旧账号突然激活**：长期不更新的账号突然发布推广内容
9. **封面图杂乱无章**：无统一风格，影响用户体验
10. **短时间内密集发布**：一次性更新多篇笔记，被视为刷屏
11. **内容跨度太大**：包括过多广告，与粉丝主题不符
12. **露出违禁方式**：在笔记/个人资料中出现第三方平台关键词
13. **伪造粉丝数据**：刷粉丝数量，使用水军等行为
14. **认证与内容不符**：账号认证领域与实际发布内容不相关
15. **直接广告营销**：直接推广产品，缺乏内容创意
16. **被多次举报**：进入人工审核，流量受限
17. **设备登录频繁**：同一设备频繁切换多个账号
18. **个人信息含广告**：头像/连系方式、logo涉及广告内容
19. **频繁修改账号信息**：经常更换账号名、头像等
20. **内容被多次违规**：多次内容违规累积

## 可执行工作流

### 工作流 1：新手首次发布笔记

**适用场景：** 第一次在小红书发布笔记的新手用户

**执行步骤：**

1. 下载小红书APP，使用手机号注册并完善个人资料（头像、昵称、简介）
2. 养号3-7天：每天浏览、点赞、评论同赛道内容
3. 找10个同赛道低粉爆文博主，拆解封面、标题、结构
4. 在创作者中心 → 笔记灵感中选择热门话题
5. 准备素材：拍摄/制作图片或视频，用修图软件制作封面
6. 通过创作者中心 → 笔记灵感 → 去发布（不要直接点+号）
7. 上传素材，编辑内容，撰写标题（痛点→分析→效果结构）
8. 添加10个相关标签和话题
9. 如果使用了AI辅助，在高级选项中标注
10. 点击发布，发布后2小时内积极互动

**预期产出：** 一篇符合平台规范的高质量首发笔记

### 工作流 2：发布后自检流程

**适用场景：** 笔记发布后检查是否一切正常

**执行步骤：**

1. 发布后30分钟，搜索笔记标题关键词，确认是否被收录
2. 检查小眼睛（浏览量）是否正常（新号200-500为基础曝光）
3. 检查是否收到平台违规通知
4. 如浏览量异常低（<100），逐项排查20个危险行为清单
5. 2小时内在评论区积极互动，引导高质量评论
6. 一周后复盘数据：阅读量、互动率、收藏率

**预期产出：** 确认笔记正常展示，及时发现并修复问题

### 工作流 3：账号限流恢复

**适用场景：** 发现账号被限流后的恢复操作

**执行步骤：**

1. 停止一切违规行为
2. 在「帮助与客服」→「笔记申诉」或「账号安全中心」提交申诉
3. 提供真实有效的证明材料（原创证明、资质证书等），7天内提交
4. 连续发布7天优质原创内容
5. 参与平台官方活动（话题挑战、创作者学院课程）获取额外流量扶持
6. 主动删除违规笔记，清理评论区违规内容
7. 等待3个工作日内的申诉结果

**预期产出：** 账号权重逐步恢复，流量回归正常

## 常见误区与避坑

1. **误区：** 直接点底部"+"号就能发布笔记
   **真相：** 通过创作者中心 → 笔记灵感入口发布，流量会更好，因为可以蹭到热门话题流量

2. **误区：** AI生成的内容不标注也没关系
   **真相：** 2026年起AI内容标注为强制要求，未标注首次限流7天，三次降权1个月

3. **误区：** 刷赞刷评论能提升流量
   **真相：** 平台精准识别"有效互动"，机械互动不仅不加分反而扣分

4. **误区：** 新号没有流量优势
   **真相：** 百万博主和新号的初始曝光机制完全一样（200-500基础曝光），新号30天内还有额外20%流量扶持

5. **误区：** 发布越多越好
   **真相：** 短时间密集发布会被判定为刷屏，建议日更1篇或隔天更新

6. **误区：** 在评论区/简介里引导加微信
   **真相：** 2026年站外引流为最严整治项，OCR可识别图片中的二维码，三次导流永久封号

7. **误区：** 笔记数据不好就删掉重发
   **真相：** 频繁删除笔记会被系统判定为账号内容质量不稳定，导致限流
